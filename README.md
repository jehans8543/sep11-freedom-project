# SEP11 Freedom Project
by **Jehan Sayeed**

## Context
I am a student at [HSTAT](https://www.hstat.org/) in the [Software Engineering Program](https://hstatsep.github.io/). The "Freedom Project" for SEP11 is a year-long project all about making something using JavaScript along with a third-party JS tool.

For my project, I chose to independently study **three.js** in order to help me make **3D model of an animal cell**. In short, [three.js](https://threejs.org/) is a JavaScript library used to create and display animated, interactive 3D computer graphics directly inside a web browser. I used three.js as the main tool and framework to create and render a 3D model of my animal cell. For other aspects of my project, I used _HTML_ and a little bit of _CSS_, to help make the popup cards of organelles. 

In terms of the topic of my freedom project,  I choose the topic of _animal cell_, the reason why I choose this topic was because I was taking AP Biology this year, and I figured that a way to help me understand what I'm learning in class was to make a animal cell myself so I would be able to visualize it and understand how it works. 

In this freedom project, we were given a bit more freedom then we had last year in my opinion, since were allowed to incorporate our previous knowledge of HTML, CSS, and now javascript. Hence, I believe that made this year's freedom project easier in a sense. 

Overall, my whole project centers around my tool _three.js_, since without it I wouldn't be able to make the framework nor my project in general.
